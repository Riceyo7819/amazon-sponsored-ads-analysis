## Business Objective
The objective of this project is to identify:
- search terms that drive profitable sales
- high-cost / low-conversion keywords
- potential opportunities for bid adjustment and negative keyword optimization

## Files
- `sample_amazon_search_term_report.xlsx` — anonymized sample dataset
- `amazon_ads_analysis.ipynb` — full analysis workflow in Jupyter Notebook
- `index.html` — final HTML report

## Key Metrics
- Impressions
- Clicks
- CTR
- CPC
- Spend
- Sales
- ACOS
- ROAS
- Conversion Rate

## Tools Used
- Python
- Pandas
- Jupyter Notebook
- Excel
- HTML Report

## Key Analysis Areas
- Data cleaning and preparation
- KPI calculation
- Search term performance evaluation
- Advertising efficiency analysis
- Optimization recommendations

## Business Value
This project demonstrates how advertising data can be used to identify profitable search terms, reduce wasted spend, and support performance marketing decisions in an e-commerce setting.

## Note
Commercial data has been anonymized or partially masked for confidentiality.
